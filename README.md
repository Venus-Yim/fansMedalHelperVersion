### 如何贡献文档  

- 安装 pnmp  
```shell
npm install -g pnpm
```  
- 克隆文档仓库  
```shell
git clone https://github.com/XiaoMiku01/fansMedalHelperVersion.git  
cd fansMedalHelperVersion
```
- 安装依赖  
```shell
pnpm install
```  
- 调试预览  
```shell
pnpm docs:dev
```  
- 打包静态文件  
```shell
pnpm docs:build
```
